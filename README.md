## How It Works

1. Customer browses the catalog from multiple vendors and adds items to cart
2. The system splits the order by vendor and assigns delivery zones
3. Each vendor receives their portion of the order and starts fulfilment
4. Delivery drivers are auto-assigned and routed for optimal efficiency
5. Payment is processed; customer tracks multi-vendor delivery in real time
6. Order completes; platform settles payments to each vendor
